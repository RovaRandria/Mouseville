Le .jar externe de JUnit 4.12 se trouve à l'adresse suivante : http://www.mediafire.com/download/zihr00ee46k297x/junit-4.12.jar
Pour exécuter le programme, il faut extraire l'archive puis il suffit d'exécuter simulationGUI.java ou de lancer "Mouseville.jar" à partir de la racine.

Informations supplémentaires :
Nous n'avons pas utilisé JLog4 dû à la présence d'un système de log dans notre programme.
JUnit a très peu été utilisé, nous avons négligé son utilité pendant la conception des différentes parties.
La javadoc n'est pas disponible dû à un problème de compilation arrivé très tard.. Tous les fichier possèdent néanmoins des commentaires javadoc.
